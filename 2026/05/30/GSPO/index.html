з»„еәҸеҲ—зӯ–з•ҘдјҳеҢ–пјҲGSPOпјүпјҡеӨ§жЁЎеһӢејәеҢ–еӯҰд№ зҡ„еәҸеҲ—зә§жј”иҝӣ

дёҖгҖҒ GRPO зҡ„еә•еұӮзјәйҷ·пјҡToken зә§йҮҚиҰҒжҖ§йҮҮж ·зҡ„ж•°еӯҰзЎ¬дјӨ

иҷҪз„¶ GRPOпјҲGroup Relative Policy OptimizationпјүйҖҡиҝҮвҖңз»„еҶ…з®— Z-ScoreвҖқжҲҗеҠҹејҖйҷӨдәҶ Critic жЁЎеһӢпјҢдҪҶеңЁеӨ„зҗҶ**й•ҝж–Үжң¬жҺЁзҗҶпјҲReasoningпјүе’Ңж··еҗҲдё“е®¶жЁЎеһӢпјҲMoEпјү**ж—¶пјҢе®ғз•ҷдёӢдәҶдёҘйҮҚзҡ„ж•°еӯҰдёҺйҖ»иҫ‘дёҚеҢ№й…ҚпјҲMismatchпјүйҡҗжӮЈгҖӮ

1. дјҳеҠҝдёҺ Ratio зҡ„йў—зІ’еәҰй”ҷдҪҚ

  • е…ЁеҸҘе…ұдә«зҡ„дјҳеҠҝпјҡеңЁеҺ»жҺү Critic еҗҺпјҢж•ҙеҸҘиҜқйҮҢжүҖжңүзҡ„ Token е…ұдә«зҡ„йғҪжҳҜеҗҢдёҖдёӘз”ұз»„еҶ…жҺ’йҳҹз®—еҮәжқҘзҡ„е…ЁеҸҘзӣёеҜ№дјҳеҠҝ A^i\hat{A}_iвҖӢAвҖӢ^вҖӢвҖӢвҖӢiвҖӢвҖӢгҖӮ
  • Token зә§зҡ„ RatioпјҡеңЁжўҜеәҰеӣһдј ж—¶пјҢGRPO еҚҙжІҝз”ЁдәҶ PPO зҡ„ Token зә§йҮҚиҰҒжҖ§йҮҮж ·жҜ”зҺҮпјҡ

    ratiot=ПҖОё(atвҲЈst)ПҖОёold(atвҲЈst)\text{ratio}_t = \frac{\pi_{\theta}(a_t|s_t)}{\pi_{\theta_{\text{old}}}(a_t|s_t)} ratioвҖӢtвҖӢвҖӢ=вҖӢПҖвҖӢОёвҖӢoldвҖӢвҖӢвҖӢвҖӢ(aвҖӢtвҖӢвҖӢвҲЈsвҖӢtвҖӢвҖӢ)вҖӢвҖӢПҖвҖӢОёвҖӢвҖӢ(aвҖӢtвҖӢвҖӢвҲЈsвҖӢtвҖӢвҖӢ)вҖӢвҖӢ

2. иҮӘеӣһеҪ’й•ҝе°ҫзҡ„ж–№е·®зҲҶзӮёпјҲзҒҫйҡҫжҖ§еҙ©жәғе…ғеҮ¶пјү

еңЁеӨ§жЁЎеһӢиҮӘеӣһеҪ’з”ҹжҲҗдёӯпјҢToken д№Ӣй—ҙеӯҳеңЁејәзғҲзҡ„жқЎд»¶жҰӮзҺҮзә§иҒ”ж•Ҳеә”гҖӮ
з”ұдәҺеүҚйқўеҸӮж•°зҡ„еҫ®е°ҸеҸҳеҠЁпјҢй•ҝе°ҫйҳ¶ж®өпјҲеҸҘе°ҫпјүToken зҡ„жҰӮзҺҮжіўеҠЁдјҡиў«еҺҶеҸІдёҠдёӢж–ҮжҢҮж•°зә§ж”ҫеӨ§гҖӮиҝҷдјҡеҜјиҮҙй•ҝе°ҫеӨ„зҲҶеҮәжһҒз«Ҝзҡ„и¶…зә§еӨ§ ratiot\text{ratio}_tratioвҖӢtвҖӢвҖӢпјҲдҫӢеҰӮй«ҳиҫҫеҮ еҚҒеҖҚпјүгҖӮ
еңЁ Backward ж—¶пјҢиҝҷдәӣдёҺж ёеҝғи§ЈйўҳйҖ»иҫ‘ж— е…ізҡ„еәҹиҜқ Token дјҡеҲҶеҲ°жҒҗжҖ–зҡ„ж–№е·®е·ЁжөӘпјҲжўҜеәҰеҷӘеЈ°пјүпјҢеҪ»еә•ж·№жІЎзңҹжӯЈзҡ„дёҡеҠЎжўҜеәҰпјҢеҜјиҮҙжЁЎеһӢеҸ‘з”ҹзҒҫйҡҫжҖ§еҙ©жәғпјҲCatastrophic CollapseпјүпјҢиЎЁзҺ°дёә Loss йЈҷеҚҮжҲ–зӢӮеҗҗд№ұз ҒгҖӮ

3. MoE жһ¶жһ„дёӢзҡ„и·Ҝз”ұйңҮиҚЎ

еңЁ Token зә§еҲ«жӣҙж–°ж—¶пјҢеҸӮж•°зҡ„иҪ»еҫ®жү°еҠЁдјҡеҜјиҮҙжҹҗйў— Token зӘҒз„¶иў«и·Ҝз”ұеҲ°дёҚеёёз”Ёзҡ„ Expert иә«дёҠгҖӮиҜҘ Token зҡ„ ratiot\text{ratio}_tratioвҖӢtвҖӢвҖӢ зһ¬й—ҙжҡҙж¶ЁпјҢдјҡеңЁжІЎжңүд»»дҪ•дҝқжҠӨжңәеҲ¶зҡ„жғ…еҶөдёӢзӣҙжҺҘжҠҠиҜҘдё“е®¶вҖңи®ӯеәҹвҖқпјҢеј•еҸ‘ MoE и·Ҝз”ұеқҚеЎҢгҖӮ

дәҢгҖҒ GSPO зҡ„ж ёеҝғдјҳеҢ–пјҡе…ЁзӣҳеӣһеҪ’вҖңеәҸеҲ—зә§пјҲSequence-levelпјүвҖқ

дёәдәҶеҪ»еә•и§ЈеҶідёҠиҝ°зЎ¬дјӨпјҢQwen еӣўйҳҹжҸҗеҮәдәҶ GSPOпјҲGroup Sequence Policy OptimizationпјүгҖӮе…¶ж ёеҝғе“ІеӯҰжҳҜпјҡж—ўз„¶еҘ–еҠұе’ҢдјҳеҠҝжҳҜй’ҲеҜ№ж•ҙзҜҮеәҸеҲ—пјҲSequenceпјүзҡ„пјҢйӮЈд№ҲйҮҚиҰҒжҖ§йҮҮж ·е’ҢеүӘиЈҒд№ҹеҝ…йЎ»еҚҮзә§еҲ°еәҸеҲ—зә§пјҒ

1. дјҳеҢ–зӮ№дёҖпјҡеәҸеҲ—зә§йҮҚиҰҒжҖ§йҮҮж ·пјҲSequence-level Ratioпјү

GSPO зӣҙжҺҘзү©зҗҶи¶…еәҰдәҶ Token зә§еҲ«зҡ„жҰӮзҺҮжҜ”еҜ№пјҢж”№з”Ёж•ҙеҸҘиҜқзҡ„ж•ҙдҪ“дјјз„¶жҰӮзҺҮжҜ”еҖјпјҲSequence Likelihood Ratioпјү Si(Оё)S_i(\theta)SвҖӢiвҖӢвҖӢ(Оё)пјҡ

Si(Оё)=ПҖОё(YiвҲЈX)ПҖОёold(YiвҲЈX)=exp(вҲ‘t=1TlogПҖОё(ytвҲЈx,y<t)вҲ’вҲ‘t=1TlogПҖОёold(ytвҲЈx,y<t))S_i(\theta) = \frac{\pi_{\theta}(Y_i|X)}{\pi_{\theta_{\text{old}}}(Y_i|X)} = \exp \left( \sum_{t=1}^{T} \log \pi_{\theta}(y_t|x, y_{<t}) - \sum_{t=1}^{T} \log \pi_{\theta_{\text{old}}}(y_t|x, y_{<t}) \right) SвҖӢiвҖӢвҖӢ(Оё)=вҖӢПҖвҖӢОёвҖӢoldвҖӢвҖӢвҖӢвҖӢ(YвҖӢiвҖӢвҖӢвҲЈX)вҖӢвҖӢПҖвҖӢОёвҖӢвҖӢ(YвҖӢiвҖӢвҖӢвҲЈX)вҖӢвҖӢ=exp(вҖӢt=1вҖӢвҲ‘вҖӢTвҖӢвҖӢlogПҖвҖӢОёвҖӢвҖӢ(yвҖӢtвҖӢвҖӢвҲЈx,yвҖӢ<tвҖӢвҖӢ)вҲ’вҖӢt=1вҖӢвҲ‘вҖӢTвҖӢвҖӢlogПҖвҖӢОёвҖӢoldвҖӢвҖӢвҖӢвҖӢ(yвҖӢtвҖӢвҖӢвҲЈx,yвҖӢ<tвҖӢвҖӢ))

йҖҡиҝҮеңЁ Log з©әй—ҙеҜ№е…ЁеҸҘ Token жҰӮзҺҮжұӮе’ҢпјҲиҝһд№ҳеҗҺе№іеқҮпјүпјҢй•ҝе°ҫ Token зҡ„еұҖйғЁеү§зғҲйңҮиҚЎеңЁж•ҙдҪ“жҖ»иҙҰйҮҢиў«иҮӘеҠЁеҜ№еҶІе’Ңзү©зҗҶжҠ№е№іпјҢеҪ»еә•ж¶ҲйҷӨдәҶжўҜеәҰзҡ„зҙҜз§Ҝж–№е·®гҖӮ

2. дјҳеҢ–зӮ№дәҢпјҡеәҸеҲ—зә§дёҖеҲҖеҲҮеүӘиЈҒпјҲSequence-level Clippingпјү

GSPO дёҚеҶҚеңЁжҜҸдёӘ Token еӨ„еҗ„з®—еҗ„зҡ„ ClipпјҢиҖҢжҳҜзӣҙжҺҘеҜ№ж•ҙеҸҘиҜқзҡ„ж•ҙдҪ“жјӮз§»зЁӢеәҰ Si(Оё)S_i(\theta)SвҖӢiвҖӢвҖӢ(Оё) з»ҹдёҖе®һиЎҢдёҖеҲҖеҲҮпјҡ

LGSPO(Оё)=1NвҲ‘i=1Nmin(Si(Оё)A^i,clip(Si(Оё),1вҲ’Пө,1+Пө)A^i)L^{\text{GSPO}}(\theta) = \frac{1}{N} \sum_{i=1}^{N} \min \left( S_i(\theta)\hat{A}_i, \text{clip}(S_i(\theta), 1-\epsilon, 1+\epsilon)\hat{A}_i \right) LвҖӢGSPOвҖӢвҖӢ(Оё)=вҖӢNвҖӢвҖӢ1вҖӢвҖӢвҖӢi=1вҖӢвҲ‘вҖӢNвҖӢвҖӢmin(SвҖӢiвҖӢвҖӢ(Оё)вҖӢAвҖӢ^вҖӢвҖӢвҖӢiвҖӢвҖӢ,clip(SвҖӢiвҖӢвҖӢ(Оё),1вҲ’Пө,1+Пө)вҖӢAвҖӢ^вҖӢвҖӢвҖӢiвҖӢвҖӢ)

еҰӮжһңж•ҙзҜҮдҪңж–Үзҡ„ж•ҙдҪ“жјӮз§»и¶…еҮәдәҶе®үе…ЁеҢәпјҢе…ЁеҸҘеңЁеҸҚеҗ‘дј ж’ӯж—¶ж•ҙйҪҗж–ӯз”өгҖӮиҝҷйҒҝе…ҚдәҶ GRPO дёӯеӣ еұҖйғЁ Token з•ёеҸҳеј•иө·зҡ„жўҜеәҰеҸҳеҪўпјҢи®©дёҖж•ҙеҸҘиҜқзҡ„жүҖжңү Token ж•ҙж•ҙйҪҗйҪҗең°е…ұдә«е№іж‘ҠеҗҺзҡ„зәҜеҮҖжўҜеәҰгҖӮ

3. дјҳеҢ–зӮ№дёүпјҡеӨ©з„¶йҖӮй…Қ MoE ејәеҢ–еӯҰд№ 

з”ұдәҺеј•е…ҘдәҶ Sequence зә§еҲ«зҡ„е…ЁеұҖдјјз„¶дј°и®ЎпјҢMoE жЁЎеһӢеңЁ RL иҝҮзЁӢдёӯдёӘеҲ« Token зҡ„дё“е®¶и·Ҝз”ұдәӨжӣҝжіўеҠЁпјҢдјҡиў«е…ЁеұҖжұӮе’Ңж— зјқзЁҖйҮҠгҖӮGSPO е…Ғи®ё MoE жһ¶жһ„еңЁдёҚйңҖиҰҒд»»дҪ•еӨҚжқӮи·Ҝз”ұдҝқжҠӨзј“еҶІеҢәпјҲRouting Replayпјүзҡ„еүҚжҸҗдёӢпјҢе®һзҺ°жһҒеәҰе№іж»‘гҖҒзЁіе®ҡзҡ„ејәеҢ–еӯҰд№ еҫ®и°ғгҖӮ
